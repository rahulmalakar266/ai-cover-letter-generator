# AI Cover Letter Generator

An AI-powered web application that generates professional and ATS-friendly cover letters using OpenRouter AI.

## Features

- AI-powered cover letter generation
- ATS-friendly output
- Copy to Clipboard
- Loading state while generating
- Secure API key using `.env`
- Responsive UI

## Tech Stack

- Vite
- TypeScript
- HTML
- CSS
- OpenRouter API

## Installation

```bash
git clone https://github.com/rahulmalakar266/ai-cover-letter-generator.git
cd ai-cover-letter-generator
npm install
```

Create a `.env` file:

```env
VITE_OPENROUTER_API_KEY=YOUR_API_KEY
```

Run the project:

```bash
npm run dev
```
